# NetPractice

## Descrição

O **NetPractice** é um projeto prático de aprendizado cujo objetivo é introduzir os conceitos fundamentais de redes de computadores.  
O projeto tem como foco a compreensão de como uma rede funciona por meio da configuração correta de endereços IP, máscaras de sub-rede, gateways, roteadores e switches.

Através de uma interface de treinamento interativa, o estudante deve corrigir diagramas de redes não funcionais, aplicando corretamente os princípios de endereçamento TCP/IP e roteamento.  
As redes utilizadas no projeto são **simuladas**, servindo exclusivamente para fins educacionais.

## Instruções

### Execução da interface de treinamento

1. Faça o download dos arquivos disponíveis na página do projeto NetPractice.
2. Extraia os arquivos em qualquer diretório de sua preferência.
3. Abra o arquivo `index.html` em um navegador web.

Ao abrir o arquivo, a interface de treinamento do NetPractice será exibida.

### Utilização da interface

- Insira seu **login da 42** no campo indicado para utilizar sua configuração pessoal.
- Alternativamente, utilize a aba **evaluation** para gerar configurações aleatórias semelhantes às usadas em avaliações.
- O projeto contém **10 níveis de treinamento**, cada um apresentando um diagrama de rede inicialmente não funcional.
- Para cada nível:
  - Modifique apenas os campos não sombreados.
  - Utilize o botão **Check again** para verificar se a configuração está correta.
  - Após concluir o nível com sucesso, utilize o botão **Get my config** para exportar o arquivo de configuração.

⚠️ **Importante:**  
Antes de avançar para o próximo nível, é obrigatório exportar o arquivo de configuração correspondente.

### Requisitos de submissão

- Devem ser exportados **10 arquivos de configuração**, sendo **um por nível**.
- Todos os 10 arquivos devem estar localizados **na raiz do repositório Git**.
- Apenas os arquivos presentes no repositório serão avaliados durante a defesa.

Durante a defesa:
- Será necessário resolver **três níveis aleatórios**, dentro de um tempo limitado.
- O uso de ferramentas externas não é permitido.
- Apenas uma calculadora simples, como `bc`, é tolerada.

## Recursos

### Conceitos de redes estudados

Este projeto aborda explicitamente os seguintes conceitos de redes:

- Endereçamento TCP/IP
- Endereços IPv4
- Máscara de sub-rede
- Gateway padrão
- Roteadores
- Switches
- Interfaces de rede
- Camadas do modelo OSI
- Princípios básicos de roteamento
- Segmentação de redes

- Subject oficial do projeto NetPractice (currículo da 42)

- Tutoriais introdutórios sobre endereçamento e roteamento de redes

### Uso de Inteligência Artificial

Ferramentas de IA foram utilizadas **apenas como suporte ao aprendizado**, em conformidade com as diretrizes da 42, para:

- Esclarecimento de conceitos teóricos de redes (TCP/IP, máscara de sub-rede, gateway, camadas OSI)
- Auxílio na interpretação dos logs exibidos pela interface de treinamento
- Organização e estruturação da documentação (README)

Nenhuma configuração de rede foi copiada diretamente de respostas geradas por IA.  
Todas as soluções submetidas foram desenvolvidas manualmente, compreendidas e revisadas criticamente.

A validação do aprendizado foi reforçada por meio de discussões e revisões com colegas.
